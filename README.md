# toby-android
A Toby Android application.

## What is it?

This Android application allows you to make your device a Toby bot! Just sign in as a standard bot, and you'll be able to send it commands from your other bots.

Abilities:
- Text-to-speech
- Speech recognition
- Copy-to-clipboard
- Set volume
- Take picture
- Shake detector
- Light detector
